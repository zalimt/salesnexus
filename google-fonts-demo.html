Usage Examples

In SCSS:

// Using font family variables
.my-text {
    font-family: $font-lexend-deca;
    font-weight: $font-weight-medium;
}

// Using utility classes in HTML
<p class="font-lexend font-semibold">Text content</p>
<h1 class="font-caladea font-bold">Heading</h1>

In CSS:

.my-heading {
    font-family: 'Caladea', serif;
    font-weight: 700;
}

.my-body-text {
    font-family: 'Lexend Deca', sans-serif;
    font-weight: 400;
}

Available Utility Classes:

  • .font-lexend - Applies Lexend Deca font
  • .font-caladea - Applies Caladea font
  • .font-light - Font weight 300
  • .font-regular - Font weight 400
  • .font-medium - Font weight 500
  • .font-semibold - Font weight 600
  • .font-bold - Font weight 700
